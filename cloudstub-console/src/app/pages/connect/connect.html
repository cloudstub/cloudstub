Configure the CloudStub instance to connect to

The base URL of the CloudStub API port (default: 4567)

@if (api.connected()) { Connected to {{ api.host() }} } @else { Not connected }

How to start CloudStub

java -jar cloudstub-standalone.jar

By default, the mock port is 4566 and the API port is 4567.

Override with --api-port=<n> or CLOUDSTUB_API_PORT env var.
